to get this running:

1. install Canary, the semi-dev release of Chrome: https://tools.google.com/dlpage/chromesxs
2. in canary, go to chrome://flags and enable "experimental"
3. restart canaray
4. in Canary -> settings -> extensions -> developer mode --> on, then "load unpacked extension"
5. select local path of uistsic/chrome-app
6. open new tab in canary and the app should be there!
